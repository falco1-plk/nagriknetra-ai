# 🌿 NagrikNetra AI

AI-powered citizen environmental reporting platform built for the **Build with AI: Code for Communities Hackathon**.

## Features

- AI-powered pollution detection using Gemini Vision
- Image upload and analysis
- Structured environmental inspection report
- OpenStreetMap integration
- Risk score generation
- Citizen and municipal recommendations

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Gemini API
- Leaflet
- OpenStreetMap
